=== handWrite post for iPad ===Contributors: yutaka miyawakiTags: ipad,canvasRequires at least:3.0Tested up to:3.12Stable tag:0.9.0handwrite post with canvas(html5) from iPad.html5のキャンバスを用いて、iPadから手書きの記事を投稿できます。== Description ==this plugin can be used only with iPad.you can post handwrite image file(jpeg) easiry.rotate your iPad to landscape Style and use.このプラグインはiPadからのみ利用することができます。とても簡単に手書き画像を投稿することができます。iPadを横向けにして利用してください。== Installation ==1. Upload the `handwrite-post-for-ipad` folder to the `/wp-content/plugins/` directory. `handwrite-post-for-ipad`を`/wp-content/plugins/`ディレクトリにアップします。2. Activate `handwrite-post-for-ipad` through the 'Plugins' menu in WordPress.  Wordpressのプラグインメニューから`handwrite-post-for-ipad`を有効化します。3. Go to the `configure` control panel, and select `handwrite post` , and configure the options to your liking.  If you want to keep the defaults, just leave everything as-is.  設定メニューから`handwrite post`を選択します・設定値をお好みで設定してください。そのままでよければ何も変更せずおいてください。4. Go to `Post` control panel,and select `handwrite post` and use. 投稿メニューから`handwrite post`を選択します、利用します。== Changelog === 0.8 =* Released 2011/04/24= 0.9 =* Released 2011/05/07* 投稿ページの修正* カラー投稿対応* readmoreの簡易挿入ボタンの追加* キャンバスのスワイプ選択対応== Frequently Asked Questions === 画像の投稿枚数は変更できますか? =設定画面で投稿画面内で選択できるキャンバスの数を設定できます。また、利用しなかったキャンバスからは画像が生成されないため、投稿には反映されません。= 画像はギャラリーに登録されますか? =現バージョン(0.9)では手書き画像はギャラリーには登録されません。また、いつの段階でそれを行うかも決まっていません。== Screenshots ==1.
